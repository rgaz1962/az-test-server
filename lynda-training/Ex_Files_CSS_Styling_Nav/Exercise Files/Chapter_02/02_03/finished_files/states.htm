Styling link states

CSS allows us to style various states of links by using what are known as pseudo-class selectors. Pseudo-class selectors allow us to select elements based on properties such as links states or user interactions.

Dynamic pseudo-class examples

  • Anchor element with no href
  • :link targets unvisited links with an href attribute
  • :visited targets visited links
  • :hover targets links when hovered over
  • :active targets currently active links
  • :focus targets links that have focus
