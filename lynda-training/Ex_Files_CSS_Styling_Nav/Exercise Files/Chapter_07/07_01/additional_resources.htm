Additional resources

Some handy links to help you learn more about, and become more productive in, CSS.

CSS Current Work Page

Current listing of all W3C CSS specifications. Frequently updated, and provides a road map of where CSS is heading. Pay particular attention to the Flex Box, CSS Exclusions, Grid Layout, and CSS Templates specs. While not currently widely supported, and still in the early stages of development, these specifications provide a look at how we'll control page layout in the future.

WebAIM's guide to link accessibility

Great resource for learning more about making links accessible to all users.

Smashing Magazine's "The Definitive Guide to Styling Web Links"

Fantastic article by Lee Munroe that gives a thorough overview of best practices and considerations for styling links.

WebAIM Color Contrast Checker

Useful tool for comparing the contrast of multiple colors to make sure your color schemes remain accessible

Ultimate CSS Gradient Generator

Fantastic tool for quickly creating and editing CSS gradients. Will generate the proper syntax for you, and has options for exactly how you want the syntax structured.

CSS3 Generator

Great tool for experimenting with values and generating syntax for several new CSS3 features.

Adobe Kuler

One of my favorite online tools. Allows you to generate and tweak color schemes based on various color schemes.

CSS Sprites

Online tool for generating CSS Sprites and the resulting CSS code for using them.
